# Tridados
Plataforma web para jogadores e mestres do sistema 3DeT Victory. O Tridados centraliza tudo que uma mesa precisa: criação de fichas, gerenciamento de NPCs, campanhas e conteúdo homebrew.


# Funcionalidades

Área dos Jogadores
Criação de fichas de personagem com distribuição de pontos
Escolha de perícias, vantagens e desvantagens
Visualização de fichas criadas
Área dos Mestres
Criação e gerenciamento de NPCs
Criação de campanhas
Visualização das fichas dos jogadores
Regras
Consulta às regras base do 3DeT Victory diretamente na plataforma
Homebrew
Criação de arquétipos, vantagens, perícias e regras personalizadas

# Tecnologias

Back-end | Python (Flask)

Front-end | HTML, CSS, JavaScript

Armazenamento | JSON

Deploy | Render


# Status

Como rodar localmente

# Clone o repositório

git clone https://github.com/hilen-dev/Tridados.git

cd Tridados

# Instale as dependências

pip install -r requirements.txt

# Inicie o servidor

python app.py

Acesse em http://localhost:5000

# Sobre o 3DeT Victory
O 3DeT Victory é um sistema de RPG de mesa brasileiro criado por Marcelo Cassaro. O Tridados é um projeto não oficial e de uso livre, sem fins lucrativos.

Todos os direitos do sistema pertencem aos seus respectivos criadores.
